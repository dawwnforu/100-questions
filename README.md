# 100 Questions About Me (100个问题 - 关于我)

An interactive single-page web app that presents 100 personal questions across categories. Browse, answer, and reflect — designed for self-discovery and sharing with friends.

## Features

- 100 questions organized by category
- Progress tracking with a visual progress bar
- Copy individual answers or export all at once
- Responsive design, works on mobile and desktop
- Clean, minimal interface with a warm aesthetic

## Usage

Open `index.html` in any browser. No build step, no dependencies.

## Structure

- `index.html` — the entire app (HTML + CSS + vanilla JS)
